# CoffeeMasiha

A plain React and Vite website for CoffeeMasiha by Mohnish.

## Prerequisites

- Node.js `>=22.13.0`

## Quick Start

```bash
npm install
npm run dev
npm run build
```

## Useful Commands

- `npm run dev`: start local development.
- `npm run build`: type-check and create a production build in `dist/`.
- `npm run preview`: preview the production build locally.
- `npm run lint`: run ESLint.

## Project Shape

- `src/App.tsx`: page content.
- `src/BrewScrollScene.tsx`: animated 3D coffee ritual scene.
- `src/styles.css`: site styling.
- `public/`: static images and favicon.
